## 팀명 : Don't Quit

#### 빅리더 AI 아카데미 에서 진행한 *코트라(대한무역투자진흥공사)* 협업 프로젝트

## 1. **프로젝트 동기**
    * HSCODE 란
      * 국제 통일 분류 체계
      * 최대 10자리로 이루어져있다.
      * 관세율과 밀접한 관계가 있다.
      * 수출입 통관 서류에서 매우 중요한 역할을 한다.
    * 문제 현황
      * 현재 KOTRA의 HSCODE 검색 사이트에서 실거래 품명으로 검색시 원하는 HSCODE 결과를 얻기 어렵다.

## 2. Needs & Solution
   * 실제 품목 분류 사례 데이터를 바탕으로 현장용어로도 손쉽게 HSCODE 찾을 수 있다.
